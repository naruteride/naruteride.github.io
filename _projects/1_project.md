# 웹 포트폴리오 프로젝트

이 프로젝트는 React를 사용하여 개발한 반응형 웹 포트폴리오 사이트입니다.

## 주요 기능

- 반응형 디자인
- 다크 모드 지원
- 애니메이션 효과
- 프로젝트 갤러리
- 컨택트 폼

## 사용 기술

- React
- Styled Components
- Framer Motion
- Firebase
